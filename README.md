# sprout-ticket (forked from example-node)
簡單的購物網站，以nodejs, express, mongodb, mongoose建構而成。

## Function
1. Register
 
2. Login

3. Add Item

4. Add to shopcart


## Start

1. Setup mongodb (easy for docker)

2. `node app.js`

http://0.0.0.0:3000
